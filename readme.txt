FluxBB Patcher 1.1

This script patches FluxBB files using modification readme.txt file.

1. Upload patcher.php file and patcher directory to FluxBB root directory.
2. Run FORUM_URL/patcher.php in your browser.


Adding new modifications
1. Make new directory in patcher/mods/ with name of the modification.
2. Extract files included in modification archive to directory created before.
